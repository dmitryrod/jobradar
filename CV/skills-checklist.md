# Skills Checklist

## Сильные стороны
- Prompt engineering под бизнес-задачи.
- AI-агенты: LangChain, LangGraph, Flowise.
- Workflow automation: n8n, MCP, webhooks, REST API.
- RAG: ingestion, chunking, embeddings, vector search, hybrid search, reranking.
- FastAPI для AI/backend-сервисов.
- Docker, Linux, деплой и production-обвязка.
- Интеграции: Telegram, Bitrix24, Google Sheets.
- Observability и quality для LLM: LangFuse, regression checks, LLM-as-a-judge.

## Есть опыт
- Qdrant, Pinecone, Supabase pgvector, LightRAG.
- Postgres, MongoDB, Redis, Supabase.
- GitHub Actions, Nginx.
- Claude, GPT, Gemini, Ollama.
- JavaScript для интеграционных и прикладных задач.
- Puppeteer, Browser Automation Studio.
- Cursor IDE + Claude Code как AI-native workflow разработки.

## Базово / при необходимости
- HuggingFace.
- Make.
- Дообучение и эксперименты вокруг моделей - по задаче, если это не уводит от основного product scope.

## Не мой фокус
- Обучение foundation models с нуля.
- Чисто академический ML / Data Science без привязки к продукту и интеграции.
- Heavy frontend или мобильная разработка как основная роль.
