# Проекты и ссылки

## Типы проектов
- Корпоративные AI-ассистенты и внутренние knowledge-base search решения.
- RAG-системы для поиска по документам, инструкциям, FAQ и внутренним базам знаний.
- AI-агенты и workflow-автоматизация для контента, обработки данных и квалификации лидов.
- Backend/API-сервисы для интеграции LLM-логики в существующие процессы компании.
- Интеграции с Telegram, Bitrix24, Google Sheets, REST API и webhook-сценариями.

## Что могу показать
- Краткие описания кейсов и стеков: см. соседние файлы в `CV/`.
- Публичные ссылки на часть работ: по запросу, если проект не под NDA.
- Непубличные проекты: могу описать архитектуру, ограничения, роль и принятые решения без раскрытия конфиденциальных деталей.
- GitHub / демо / примеры кода: по запросу и в зависимости от режима доступа по конкретному проекту.

## Как лучше запрашивать примеры
- Если важен RAG: покажу релевантные примеры поиска, чанкинга, retrieval и evaluation.
- Если важны агенты: покажу сценарии с LangChain / LangGraph / n8n / MCP.
- Если важен backend: покажу FastAPI, Docker, интеграции и production-обвязку.
